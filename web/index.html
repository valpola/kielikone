Options
Documentation

Use the prompt to translate. Enter your answer, press Enter to reveal, then Enter again to auto-grade. You can also grade manually with the buttons.

The login button shows who you are when result logging is enabled. Results are stored only if you log in with a valid API key.

Shortcuts

  • Enter: reveal or auto-grade
  • f: I was correct (after reveal)
  • j: I was incorrect (after reveal)
  • n: next (after reveal)
  • Tab: next (any time)

Filtering

Use Include tags to focus on a topic, and Exclude tags to avoid items. The filter applies immediately to the quiz.

Today list

Set Daily words, then click Recompute today to pick the next list based on your results. The list is stored locally on this device.

Session target

Correct to finish controls how many correct answers are required before a word is skipped for the rest of the session.
